VideoGame DB Services

This is a Asp.Net Core 2.1 project utilizing EF Core and SQLite. Now includes Odata v.4 to make building custom REST queries easier.

Current Systems/Games include:

  • Atari 2600
  • Atari 5200
  • Atari 7800
  • Atari Lynx
  • Atari Jaguar
  • Atari Jaguar CD
  • Sony PSP
  • Sony Playstation
  • Sony Playstation 2
  • NEC TurboGrafx-16
  • NEC TurboGrafx-16 CD

Examples:

https://videogamedatabase.azurewebsites.net/api/games

https://videogamedatabase.azurewebsites.net/api/games?$filter=contains(Description,'tomb')&$select=title,description

https://videogamedatabase.azurewebsites.net/api/systems
